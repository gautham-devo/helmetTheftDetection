# 🛡️ Smart Helmet Theft Detection System

### IoT-Based Helmet Security & Tracking Solution

## 📌 Overview

The Smart Helmet Theft Detection System helps users protect their helmets using **geofencing, real-time location tracking, and instant theft alerts**.

When a helmet moves outside a user-defined safe zone, the system immediately sends a notification and allows the user to track its live location through a mobile application.

---

## ✨ Features

* 📍 Real-time Helmet Tracking
* 🛡️ Geofence-Based Theft Detection
* 🚨 Instant Theft Alerts
* 🗺️ Live Location Monitoring
* 📏 Distance Tracking from Safe Zone
* 📱 Mobile App Integration
* 🔔 Push Notifications
* 🔗 Bluetooth Low Energy (BLE) Connectivity

---

## 🔄 How It Works

1. Connect helmet to the mobile app.
2. Configure a safe zone.
3. System continuously monitors helmet location.
4. If the helmet leaves the safe zone:

   * Theft alert is triggered
   * User receives a notification
   * Live tracking starts automatically
