🏍️ Helmet Theft Detection System
The Helmet Theft Detection System is an IoT-enabled anti-theft security solution
designed to protect motorcycle helmets from being stolen when left parked on the
vehicle. By combining hardware-in-the-loop sensors (to track the physical lock
status), microcontrollers (to manage connectivity), and computer vision (to
verify tampering), this system instantly triggers a localized siren and dispatches
real-time security alerts with GPS coordinates to the user's mobile device if a theft
