# 🛡️ Smart Helmet Theft Detection System

A smart anti-theft solution that helps users protect and track their helmets using geofencing, real-time location monitoring, and instant mobile alerts.

The system continuously monitors the helmet's location and immediately notifies the user whenever the helmet moves outside a predefined safe zone.

---

## 📌 Overview

Helmet theft is a common problem in colleges, offices, malls, and public parking areas.

This project provides a smart security solution by allowing users to:

- Connect their helmet to the mobile application
- Create a customizable safe zone
- Monitor helmet status in real time
- Receive theft alerts instantly
- Track helmet location on a map

## 🔄 How It Works

1. User connects the helmet to the mobile application.
2. User creates a safe zone around the parking location.
3. The system continuously monitors the helmet position.
4. If the helmet remains inside the safe zone:

   - Status shows **SECURED**
   - No alerts are generated

5. If the helmet leaves the safe zone:

   - User receives an instant notification
   - Status changes to **OUT OF ZONE**
   - Helmet location is displayed on the map
   - Distance from safe zone is calculated

6. User can track the helmet location in real time.
